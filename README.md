# Hafidz Ubaidillah — Portfolio Website

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

**Live:** https://hafidzubaidllah-sxity.github.io/hafidz-portfolio

---

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- React Icons
- React Type Animation
- Vite

## Features

- Dark monochrome design
- Typing animation hero
- Animated skill progress bars
- Framer Motion scroll animations
- Glassmorphism cards
- Floating particles background
- Responsive (mobile / tablet / desktop)
- CV download
- Sections: Hero, About, Skills, Experience, Achievement, Projects, Certificates, AI Workflow, Contact

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
